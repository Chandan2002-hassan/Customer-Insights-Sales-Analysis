Customer Insights & Sales Analysis
📌 Project Overview

This project focuses on analyzing customer shopping behavior and sales data to generate meaningful business insights. It helps in understanding customer segments, purchasing patterns, and revenue trends to support better decision-making.

📂 Dataset Details
Total Records: 3,900
Columns: 18
Includes:
Customer demographics (Age, Gender, Location, Subscription Status)
Purchase details (Item, Category, Amount, Season, Size, Color)
Shopping behavior (Discounts, Frequency, Ratings, Shipping Type)
🛠️ Tools & Technologies
Python (Pandas, Data Cleaning)
PostgreSQL (SQL Analysis)
Power BI (Dashboard & Visualization)
🔍 Data Processing
Cleaned and preprocessed raw data
Handled missing values using median method
Created new features like age groups and purchase frequency
Loaded data into PostgreSQL for structured analysis
📊 Key Analysis
Sales and revenue analysis by gender
Identification of high-value customers
Top-performing products based on ratings
Shipping type impact on purchase behavior
Subscriber vs non-subscriber comparison
Customer segmentation (New, Returning, Loyal)
Revenue contribution by age groups
📈 Dashboard

An interactive Power BI dashboard was created to visualize:

Total sales and customer metrics
Category-wise performance
Customer segmentation
Subscription insights
💡 Key Insights
Certain customer segments contribute more to revenue
Discount strategies influence purchasing behavior
Loyal customers drive consistent sales
Younger age groups show higher engagement
Shipping type impacts customer spending
🚀 Business Recommendations
Enhance customer retention through loyalty programs
Optimize discount strategies
Focus marketing on high-performing segments
Promote top-selling products
